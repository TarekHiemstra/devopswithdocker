## Smarter way
Include the curl installation in the Docker run command:

```
$ docker run --rm -it --name 1.4 ubuntu sh -c 'apt-get update; apt-get install curl; while true; do echo "Input website:"; read website; echo "Searching.."; sleep 1; curl http://$website; done'
```
